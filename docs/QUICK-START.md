# 快速开始

## 这份 fork 适合什么情况
- 想先用少量规则快速开始
- 想按场景找常用规则
- 想优先使用聚合规则而不是一个个单独找

## 最短路径
1. 先看 [常见组合](COMBOS.md)
2. 再看 [归类索引（扩展版）](CATEGORIES.md)
3. 如果是 AI 场景，优先尝试聚合规则
4. 最后按客户端接入对应规则

## Loon
- 在索引页里点对应的 Loon 链接
- AI 场景优先考虑 `AI.lsr` 这类聚合规则
- 导入后先少量启用

## Clash
- 在索引页里点对应的 Clash 链接
- 把链接写入配置文件
- 分阶段增加规则，不要一次堆太多
