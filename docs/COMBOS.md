# 常见组合推荐

本页用于快速给出一组可直接上手的搭配建议，方便你少量开始、逐步增加。

## 1. 最小可用组合
适合先跑起来：
- China
- ChinaASN
- ChinaMedia
- Cloudflare
- GitHub

## 2. 开发常用组合
适合开发、AI、协作场景：
- GitHub
- Cloudflare
- Docker
- Claude
- OpenAI
- JetBrains

## 3. 社交通讯组合
适合聊天、社区与信息获取：
- Telegram
- Discord
- Reddit
- X(Twitter)
- Facebook

## 4. 媒体娱乐组合
适合视频、音乐与流媒体：
- YouTube
- Netflix
- Spotify
- Disney
- TikTok

## 5. 国内优先组合
适合以国内服务为主的日常使用：
- China
- ChinaASN
- ChinaDNS
- ChinaIPs
- ChinaMedia

## 使用建议
- 先从“最小可用组合”开始。
- 再按实际需求增加“开发 / 社交 / 媒体”分类。
- 不建议第一次就叠加过多规则。
