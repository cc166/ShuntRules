# 自定义合集入口

这页放你自己的合集规则入口，当前以 OpenClash / Loon 模板可直接接入为目标。

## 常用合集
- AI
- Communication
- SocialMedia
- GitHub
- TikTok
- YouTube
- Netflix
- ForeignMedia
- ForeignEcommerce
- GoogleFCM
- Google
- Apple
- Microsoft
- Game
- Steam
- Speedtest

## 说明
- 这批合集优先用于模板直接接入
- `Direct` / `LAN` / `Proxy` / `ESET_China` 仍保留外部原规则源
- 其余常用分组尽量切换到你自己的 `custom/Clash/*.yaml` 与 `custom/Loon/*.lsr`
